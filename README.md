# SqlTableSchemaExporter
A MSSQL table schema exporter

 Double click to run or use argument for command line

 Argument:

 `-h` **Help**

 `-t` **Database Type**, it must be "mssql", "mysql" or "pqsql".

 `-s` **Server name or IP address**

 `-d` **Database name**

 `-u` **Account**

 `-p` **Password**
